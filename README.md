# Sentence Navigator

A series of keyboard shortcuts to select, edit and move whole sentences around an Obsidian document. 

## Installation

1. Download the zipped release file and unzip it. 
2. In Obsidian, navigate to Settings > Community Plugins. 
3. Click the folder icon above the "Installed plugins" list.  
4. Drag the unzipped release in to "plugins" folder.
5. Click the refresh icon above the "Installed plugins" list. 

You should see the plugin. Follow the same steps to update. 

## Default Shortcuts

| Command                                          | Shortcut \*                       |
| ------------------------------------------------ | --------------------------------- |
| Insert line above                                | Mod+Shift+Enter                   |
| Insert line below                                | Mod+Enter                         |
| Delete line                                      | Mod+Shift+K                       |
| Delete to end of sentence                        | Alt+Shift+Backspace               |
| Join lines                                       | Mod+J                             |
| Duplicate line                                   | Mod+Shift+D                       |
| Copy line up                                     | Ctrl+Shift+Up                     |
| Copy line down                                   | Ctrl+Shift+Down                   |
| Select word or next occurrence                   | Mod+D                             |
| Select sentence                                  | Mod+L                             |
| Reduce selection by sentence                     | Mod+Shift+L                       |
| Shift selection to next sentence                 | Mod+Shift+Right                   |
| Shift selection to previous sentence             | Mod+Shift+Left                    |
| Move sentence forward                            | Alt+Mod+Right                     |
| Move sentence back                               | Alt+Mod+Left                      |
| Select to end of sentence                        | Alt+Shift+.                       |
| Select to start of sentence                      | Alt+Shift+,                       |
| Add cursors to selection ends                    | Alt+Shift+I                       |

\* `Mod` refers to `Cmd` on macOS and `Ctrl` on Windows/Linux
